# Typemore Icon Prompt

## Prompt (for ChatGPT Image)
Design a premium macOS app icon for an app named **Typemore**, an offline speech-to-text desktop tool.

Visual concept:
- Core metaphor: a **speech waveform transforming into text lines**.
- Primary symbol: a rounded chat bubble shape containing a clean waveform that gradually becomes 3 horizontal text strokes on the right.
- Add a subtle typing cue using a small cursor/caret element near the text strokes.
- The icon should communicate: **voice input -> writing output**, fast, accurate, local-first.

Style and composition:
- Modern, minimal, geometric, highly legible at small sizes.
- macOS icon aesthetics: soft depth, tasteful highlight, refined shadow, polished but not glossy-plastic.
- Strong silhouette, centered composition, no tiny clutter.
- No letters, no words, no logo text.

Color direction:
- Background: deep teal to cyan gradient (`#0B3A46` -> `#0EA5A8`).
- Symbol: warm off-white (`#F5F7F2`) with accent mint (`#7DE2C3`) on waveform tip.
- Optional tiny accent: amber (`#F6C453`) for cursor/caret.
- Keep high contrast and color-blind-friendly separation.

Lighting and material:
- Soft top-left light source.
- Slight glass-like surface depth, subtle inner glow, gentle outer shadow.
- Clean edges; avoid noisy textures.

Output requirements:
- 1:1 square icon.
- Generate at 1024x1024.
- Provide transparent-free full icon canvas (not isolated object on transparent background).
- Ensure readability at 16x16 and 32x32 downscale.

Negative constraints:
- No microphone literal icon.
- No photorealism.
- No skeuomorphic metal/plastic parts.
- No excessive details, no thin fragile lines.
- No purple-dominant palette.

## Optional Variant Prompt (more bold)
Create a bolder variant of the same Typemore icon where the waveform-to-text transition is more dynamic and diagonal, while preserving minimalism, macOS polish, and small-size readability.
